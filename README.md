## Set-up locally and Run
* [Fork](https://github.com/shivam-sharma7/Expense-Tracker/fork) the repo
  
* Clone on local machine
 ```
  git clone https://github.com/<your_github_username>/Expense-Tracker.git
 ```
* Change directory
```
cd Expense-Tracker
```

* Install dependencies
```
npm install
```

* Run
```
npm start
```
